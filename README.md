# 📱 Student Productivity Bot

A beautiful Android app to help students stay productive!

## ✨ Features

- 📊 **Dashboard** - Real-time stats and progress tracking
- ✅ **Tasks** - Manage tasks with priorities (HIGH/MEDIUM/LOW)
- ⏱️ **Pomodoro Timer** - Focus timer with session tracking
- 📝 **Notes** - Persistent notes with search
- 📚 **Exam Prep** - Track exams with countdown
- 🔥 **Study Streak** - Stay motivated
- ☁️ **Firebase Sync** - Cloud backup and sync across devices

## 🛠️ Built With

- Kotlin

- Firebase Authentication
- Cloud Firestore
- DataStore

## 🚀 Installation

1. Download the APK from the link above
2. Enable "Unknown Sources" on your Android device
3. Open the APK and install
4. Sign up or log in to start!
